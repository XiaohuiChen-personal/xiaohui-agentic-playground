## 1. Introduction

### 1.1 Project Overview

**Sensei** is an AI-powered adaptive learning tutor built with CrewAI that provides personalized education experiences. The system uses multiple specialized AI agent crews to create courses, teach concepts, answer questions, and assess understanding.

The name "Sensei" (å…ˆç”Ÿ) means "teacher" in Japanese, reflecting the project's goal of providing wise, patient, and adaptive instruction.

### 1.2 Goals and Objectives

| Goal | Description |
|------|-------------|
| **Personalized Learning** | Adapt teaching style and pace to individual learners |
| **Comprehensive Memory** | Remember learner progress, preferences, and struggles across sessions |
| **Adaptive Assessment** | Create quizzes that target identified weak areas |
| **Interactive Experience** | Allow learners to ask questions and get clarification anytime |
| **Progress Tracking** | Provide visibility into learning journey and achievements |

### 1.3 Key Features

- **Course Generation** - AI creates complete curriculum for any topic
- **Concept-by-Concept Learning** - Structured lessons with clear explanations
- **Interactive Q&A** - Ask questions and receive contextual answers
- **Adaptive Quizzes** - Tests that focus on areas needing improvement
- **Progress Persistence** - Learning state saved across sessions
- **Multi-Memory System** - Utilizes all 5 CrewAI memory types

## Appendix: Glossary

| Term | Definition |
|------|------------|
| **Crew** | A group of AI agents that work together on related tasks |
| **Agent** | An AI entity with a specific role, goal, and backstory |
| **Task** | A specific job assigned to an agent within a crew |
| **Module** | A chapter or section within a course |
| **Concept** | A single topic or idea to learn within a module |
| **Mastery** | A measure (0-1) of how well a concept is understood |
| **Session** | A single learning session (from start to end) |
