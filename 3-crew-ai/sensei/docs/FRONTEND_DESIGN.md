## 1. Introduction

### 1.1 Overview

The Sensei frontend is a locally-running Python application that provides an intuitive interface for learners to interact with the AI-powered tutoring system. It enables users to create courses, learn concepts, ask questions, take quizzes, and track their progress.

### 1.2 Technology Choice

| Aspect | Choice | Rationale |
|--------|--------|-----------|
| **Framework** | Streamlit | Pure Python, rapid development, built-in components |
| **Runtime** | Local | Runs on localhost, no deployment needed |
| **Styling** | Streamlit native + custom CSS | Consistent look, easy customization |
| **State** | Streamlit session_state | Built-in state management |

### 1.3 Design Principles

| Principle | Description |
|-----------|-------------|
| **Simplicity** | Clean, uncluttered interface focused on learning |
| **Progress Visibility** | Always show where the learner is in their journey |
| **Responsive Feedback** | Immediate visual feedback for all actions |
| **Encouraging** | Celebrate achievements, supportive tone |
| **Accessible** | Clear typography, good contrast, keyboard navigation |
